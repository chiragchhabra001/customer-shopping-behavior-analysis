# Customer Shopping Behavior Analysis

## 📊 Project Overview
This project analyzes 3,900 customer transactions to identify revenue trends, customer loyalty patterns, and marketing effectiveness.

## 🛠 Tech Stack
- Python (Data Cleaning)
- PostgreSQL (Data Storage & Analysis)
- SQL (Business Queries)

## 📈 Key Insights
- Clothing contributes 44.73% of total revenue.
- 80% of customers are repeat buyers (10+ previous purchases).
- Medium (M) size dominates demand across categories.
- Discount usage does not significantly impact purchase value.
- Revenue is evenly distributed across seasons.

## 🎯 Strategic Recommendations
- Expand clothing inventory.
- Optimize size-based stock planning.
- Convert high-frequency non-subscribers.
- Implement targeted marketing campaigns.

## 📂 Files Included
- SQL queries for business analysis
- Python data cleaning scripts
- Business Insight Report (PDF)

## 🚀 Business Impact
Data-driven recommendations to improve customer retention, marketing ROI, and inventory efficiency.
